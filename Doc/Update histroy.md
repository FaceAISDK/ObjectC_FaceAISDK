 如果运行出现以下错误
 ```
 Swift/Integers.swift:3564: Fatal error: Not enough bits to represent the passed value  
 ```
 请在Xcode菜单Product执行clean all Issues后 再次执行pod命令升级FaceAISDK就可以了。
 ```
  pod update FaceAISDK_Core 
 ```
